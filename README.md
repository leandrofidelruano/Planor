## About Planor

Planor is an open-source workspace built for organizing ideas, managing projects, and creating structured knowledge bases.

Inspired by modern productivity tools like Notion, Planor focuses on giving users:

• Full ownership of their data
• A fast native experience
• Cross-platform support
• Flexible organization tools
• Extensible architecture for future customization

Whether you're a student, developer, freelancer, or team, Planor provides a customizable environment for planning and productivity.

---

## Features

• Rich text note-taking
• Databases and tables
• Kanban boards
• Task management
• Nested pages and workspaces
• Cross-platform sync (planned)
• Local-first architecture
• Custom templates
• Plugin-ready architecture (planned)

## Built With

Planor is built using:

• Rust — backend logic and performance-critical systems
• Flutter — cross-platform UI
• SQLite — local data storage

## Development Setup

### Requirements

Before starting, make sure you have installed:

• Flutter SDK
• Rust toolchain
• Cargo Make
• CMake
• Git

### Clone the repository

```bash
git clone https://github.com/yourusername/planor.git
cd planor
```

### Install dependencies

```bash
flutter pub get
cargo build
```

### Run the project

```bash
flutter run
```

## Why Planor?

Many productivity tools are powerful, but often come with tradeoffs:

• Closed ecosystems
• Limited customization
• Cloud dependency
• Data privacy concerns

Planor exists to provide a transparent, extensible, and developer-friendly alternative where users stay in control.

The goal is simple:

**Build a workspace that adapts to you — not the other way around.**
